# LogitechRS50TrueForceSDK
SDK for TrueForce haptic feedback on the Logitech G Racing Wheel (RS50) on Linux — no G HUB required.


> ⚠️ Reverse-engineered via USB capture analysis. This is unofficial and unsupported by Logitech. Use at your own risk.

This SDK implements TrueForce from scratch, based on full USB packet capture analysis of G HUB on Windows.
Currently only tested only on Ubuntu based Linux distros and on the RS50, but with enough captures, I am planning to extend the wheel support.
